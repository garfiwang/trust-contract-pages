信託契約-02

信託契約

立契約書人

委託人兼受益人：OOO (以下稱「甲方」)

受託人： 瑞興商業銀行股份有限公司信託部 (以下稱「乙方」)

監察人： OOO、OOO

甲方為保障本人規劃其未來包括但不限於生活、安養照護、醫療等其

他照顧需求費用之專款給付及維護財產安全等目的，茲與乙方合意訂立本

「信託契約」（下稱本契約），約定如下，以資共同遵守：

第一條 信託目的

甲方為保障本人規劃其未來包括但不限於生活、安養照護、醫療

等其他照顧需求費用之專款給付及維護財產安全等目的，將本契

約第三條約定之信託財產交付乙方，由乙方依本契約第六條及第

九條之約定，為甲方之利益管理及運用信託財產，及辦理甲方安

養專款給付及調整信託利益給付之金額或方式等事宜。

第二條 信託當事人

  1. 一、 委託人：甲方。其基本資料及聯絡方式，詳如附件一。為貫

    2. 徹及確保委託人簽訂本契約當時其為保障受益人之利益等目
    的所為之意思表示，委託人同意若其不能為有效意思表示或
    經法院為監護之宣告時，其依本契約所為指示，授權由其監
    護人行之；經法院為輔助宣告時，其依本契約所為指示，應
    經其輔助人同意後行之，惟均仍應經監察人之書面同意
    後，始得為之。
  2. 二、 受託人：乙方。
  3. 三、 受益人：本契約為自益信託，受益人即委託人甲方，甲方享

    4. 有就其本契約項下移轉交付之信託財產及其權利範圍之全部
    信託利益。
  4. 四、 信託監察人：本契約信託監察人設2人，其基本資料及聯絡

    5. 方式，詳如附件一。信託監察人就本契約約定所為之同
    意，應由信託監察人以書面共同行之。
  5. 五、 本條第三項約定之受益人，除法令或本契約另有約定外，應

經甲方（如有受法院為監護之宣告者，則為監護人）、乙方

及監察人書面合意後始得變更，其所衍生之相關稅賦，由應

稅人自行處理。

第三條 信託財產

  1. 一、 本契約之信託財產，係指本契約經各方當事人簽訂完成日

    2. 起，由甲方依本契約約定，以支票、匯款或轉帳之方式存入
    信託專戶之金錢（限新台幣，以下同），包含：
  2. (一) 甲方於本契約簽訂完成日起三十日內依本契約約定方式

    3. 存入信託專戶之金錢，共計新台幣壹仟陸佰萬元整。惟
    實際金額以委託人交付金額為準。
  3. (二) 甲方於信託財產運用及給付期間，信託專戶之金錢。
  4. 二、 甲方於信託存續期間內新增之信託財產。
  5. 三、 甲方以簽發票據方式作為交付信託財產之方式者，應俟票款

    6. 兌付後之金錢為信託財產；甲方以匯款或轉帳方式交付信託
    財產者，應匯入依本契約第六條第二項由乙方為甲方開立之
    信託專戶，並應於五日前將匯入日期及金額通知乙方。甲方
    若怠於通知乙方，因而發生損失，概由甲方自行負擔。
  6. 四、 乙方因信託財產之管理、運用、滅失、毀損或其他事由所取

    7. 得之財產權，均屬信託財產。

第四條 信託存續期間

  1. 一、 自本契約經各方當事人簽訂完成，且甲方將本契約第三條第

    2. 一項所述信託財產交付予乙方之日起生效，為期 壹 年。除本
    契約另有規定或其他法定信託消滅事由外，本信託期滿或依
    本契約第十三條約定終止事由發生時，本信託關係視為消
    滅。
  2. 二、 本契約存續期間屆滿之前，若立契約書人任一方未為反對之

    3. 意思表示，自本契約存續期間屆滿日翌日起算，自動續約一
    年，嗣後亦同。
  3. 三、 依第十三條第四項第(二)款事由終止時，甲方之繼承人應提

    4. 出死亡證明書及事由發生之相關證明文件，始得向乙方主張
    信託關係終止。

第五條 信託受益人

  1. 一、 本契約為自益信託，委託人即受益人（即為甲方） ，享有全

部信託利益，受益人不得拋棄本契約項下所得享有信託受益

權之一部或全部。

  1. 二、 受益人不得將其所得享有信託受益權之一部或全部為轉

    2. 讓、設定或質借予第三人，或為其他擔保、保證之標的。
  2. 三、 本契約存續期間，若受益人身故，受益人之信託利益歸屬權

    3. 利人應即將該事實以書面通知乙方，並檢附受益人之死亡證
    明書或其他足資證明死亡事實之文件。若信託利益歸屬權利
    人怠為上述書面通知，致受託人不知悉受益人已死亡之事
    實，而仍繼續向其履行本契約各項義務，則受託人已為之給
    付或履約行為，均生清償之效力，倘因此信託財產或其他信
    託財產歸屬權利人權益發生損害時，受託人概不負責，該信
    託財產歸屬權利人不得向受託人請求債務不履行損害賠償或
    為其他主張。

第六條 信託財產之管理及運用

  1. 一、 本契約信託財產之管理運用方法為單獨管理運用，甲方就信

    2. 託財產保留運用決定權，乙方對信託財產之管理、運用、處
    分及各項權利之行使，不具運用決定權，而應依甲方於符合
    法令規定及信託目的範圍內指示之營運範圍或方法妥善管理
    或處分信託財產。
  2. 二、 本信託財產係以乙方名義登記且保管於「瑞興商業銀行受託

    3. 信託財產專戶」（以本行新臺幣活期存款牌告利率計算利
    息，以下簡稱「信託專戶」），乙方就本信託財產之管理運
    用應造具帳簿，並分別記帳。
  3. 三、 信託財產僅限存放新台幣活期存款。
  4. 四、 甲方就信託財產為運用、管理時，應經甲方之監察人之書面

    5. 同意，並由甲方出具指示書（詳附件二，下稱「指示書」）
    對信託財產之營運範圍或方法（就交易標的、運用方式、金
    額、條件、期間等相關事項）向乙方為具體之特定運用指
    示。甲方未經其監察人書面同意致生損害，概由甲方自行負
    擔。本契約存續期間，受託人得將信託收益全數併入信託財
    產中繼續管理、運用，並為信託財產之一部分。
  5. 五、 信託財產運用所得之信託收益，受託人於所得發生年度，按

    6. 所得類別依相關稅法規定，減除成本、必要費用及耗損
    後，分別計算受益人之各類所得額，並由受益人併入當年度
    所得額申報。
  1. 六、 信託財產運用之指示，應符合相關法令規定及本信託約

    2. 定，如指示違反法令或有違反法令之虞，或乙方認為有無法
    執行或窒礙難行之虞者，乙方得立即通知甲方變更指示內
    容，若甲方不同意變更指示內容或乙方未獲任何變更指示
    者，於乙方以至少五個營業日前以書面催告後仍未獲甲方相
    關變更指示回應或甲方仍不變更原指示內容者，乙方得逕行
    終止本契約。
  2. 七、 甲方同意乙方運用信託財產時，得採下列行為：
  3. (一) 存放於乙方銀行業務部門或其利害關係人處作為存款。
  4. (二) 以信託財產與本身或其利害關係人為信託業法第二十五

    5. 條第一項規定以外之其他交易。
  5. (三) 其他經主管機關規定許可之行為。

第七條 信託財產評價方式

乙方應依中華民國信託業商業同業公會報請主管機關核定之信託

業會計處理原則計算本信託財產之淨資產價值。

第八條 信託利益分配之時期及方法

於信託存續期間，除本契約另有約定，且符合相關法令限制者

外，乙方不對甲方額外分配信託利益。

第九條 信託財產之給付

  1. 一、 甲方就信託財產之給付及運用方式，應經甲方之監察人書面

    2. 同意，並由甲方出具指示書，向乙方為指示。
  2. 二、 信託運用及給付期存續期間內，甲方因疾病、事故、支付生

    3. 前契約費用、購買醫療器材及輔具或其他與甲方照護需求相
    關事由等不定額給付需提領信託財產時，甲方得檢具指示書
    及健保特約醫療院所或其他相關機構出具之證明文件、單據
    或其他合理之說明，並經監察人之書面同意後，向乙方提出
    申請。
  3. 三、 甲方同意倘其於信託存續期間受監護宣告者，則由甲方之監

    4. 護人檢具相關事證通知乙方開始為信託財產之給付，並代為
    行使信託財產給付之約定及指示，惟仍應經監察人之書面同
    意，始得為之。甲方因其監護人怠於通知致生損害，概由甲
    方及其監護人自行負擔。

第十條 受託人之義務與責任

  1. 一、 乙方應依本契約及符合信託法、信託業法及相關法令規

    2. 定，以善良管理人之注意義務，妥善處理本契約項下一切相
    關事務。
  2. 二、 甲方之指示內容應合於相關法令規定及本契約約定，如甲方

    3. 之書面指示內容違反法令或有違反法令之虞，或乙方認為有
    無法執行或窒礙難行之虞時，乙方應依本契約第六條第六項
    約定辦理。
  3. 三、 乙方應將本契約信託財產與自有財產及其他信託財產分別記

    4. 帳管理之。
  4. 四、 乙方對於因政治、經濟、社會環境、投資運用標的本身、法

    5. 令、天災人禍、戰爭事變、武裝衝突、恐怖活動、暴動、罷
    工、天然災害及非受託人所能控制之不可抗力事由、或其他
    不可歸責於受託人，致信託財產受損失、滅失或凍結或無法
    依約分配時，乙方不負損害賠償之責。
  5. 五、 乙方就信託財產或處理信託事務所支出之稅捐、費用或負擔

    6. 之債務或遭受任何損失或損害，得逕以信託財產充之，如有
    不足抵充，乙方得向甲方請求補償，但該損失或損害係因乙
    方之故意或重大過失所致者，不在此限。
  6. 六、 乙方依本契約所負之債務，僅於信託財產之限度內負履約責

    7. 任。
  7. 七、 乙方對於本契約之相關資料資訊，除依法令規定、主管機關

    8. 或法院要求提供，或本契約另有約定外，應予保密，不得無
    故揭露予第三人。

第十一條 受託人之報酬標準、種類、計算方式、支付時期及方法

  1. 一、 設立費：甲方應於本契約簽訂日後三十日內，將設立費新台

    2. 幣（下同） 八千 元整，以匯款方式一次支付予乙方；或由
    甲方逕自信託專戶扣取。
  2. 二、 撥款作業費：甲方經取得監察人之書面同意後指示乙方依本

    3. 契約第九條約定給付信託財產時，按次計收壹仟元整，該項
    費用由甲方以匯款、轉帳、即期支票支付或由乙方逕自信託
    財產扣取。
  3. 三、 契約修改費：每次伍仟元整，由委託人經取得監察人之書面

    4. 同意後於提出契約修改申請時，另以金錢支付予受託人。惟
    本契約之修改不包括變更留存印鑑及變更個人基本資料。
  1. 四、 受託人保留依物價指數調整本契約各項收費標準之權利，如

    2. 遇調整時，受託人於生效日前六十日前以書面通知委託
    人，如委託人未於生效日前以書面提出異議，視為同意該收
    費標準之調整。
  2. 五、 本條所列相關信託報酬既經收取，不予退還，於本契約解除

    3. 或終止時，亦同。

第十二條 各項費用之負擔及其支付方法

本契約運用信託財產或處理信託事務所產生之稅捐、手續費、管

理費、負擔之債務或信託報酬如交易手續費、郵匯費、信託管理

費等其他必要之費用，或因不可歸責於受託人之事由而有與第三

人發生訴訟、仲裁或其他交涉所產生之一切費用（包括但不限於

訴訟費用、律師費用及其他處理費用）及負擔任何債務，應由甲

方負擔，受託人並得信託專戶扣取或逕以信託財產抵充之，如有

不足部分，受託人得請求甲方或歸屬權利人補償、清償債務或提

供擔保；有關受託人於本條之權利，於其權利未獲滿足前，受託

人除得停止處理本信託事務、拒絕將本信託財產依本契約約定返

還外，並得逕將信託財產依法定程序求償抵充之，甲方、其繼承

人或其他信託財產歸屬權利人均不得異議。

第十三條 信託契約之變更、解除及終止之事由

  1. 一、 信託存續期間內，除法令變更、另有規定、依主管機關命令

    2. 變更或本契約另有約定者外，本契約及附件所載之內容應經
    甲乙雙方同意，並應經監察人書面同意後，共同以書面增補
    修訂之，並視為本契約之一部分，其效力與本契約相同，但
    甲方基本資料之變更，不在此限；甲方同意如甲方經法院為
    監護之宣告，監護人得為甲方之利益代理其辦理變更，惟仍
    應經監察人之書面同意，始得為之。
  2. 二、 因信託法、信託業法及有關法令解釋之制定或變更致本契約

    3. 條款不符法令或因主管機關之命令須變更本契約內容者，甲
    方、乙方同意無條件依法令規範予以配合修訂。
  3. 三、 信託契約之解除：甲方未能依約將約定之信託財產全數交付

    4. 信託予乙方，或因其他事由致本契約簽定後，無法開始信託
    行為時，乙方得以書面通知甲方解除本契約。
  4. 四、 信託契約之終止：本契約自各方簽署完成且甲方將本契約第

    5. 三條第一項信託財產交付予乙方之日起生效，關於本契約終

止之事項，依下列各款之約定：

  1. (一) 信託目的已完成或不能完成。
  2. (二) 受益人身故，本契約即終止。
  3. (三) 信託契約存續期間屆滿前，甲乙雙方之任一方得隨時終

    4. 止本契約；但應於預定終止日前十個銀行營業日以
    前，以書面通知他方。甲方通知乙方終止時，並應經監
    察人書面同意；委託人已受監護宣告，且乙方已知悉
    者，乙方應通知監護人。甲方同意如甲方經法院為監護
    之宣告，監護人得為甲方之利益代理其終止本契約，惟
    仍應經監察人之書面同意，始得為之。但依本項第(五)
    款約定情形終止者，不在此限。
  4. (四) 信託財產已全數給付予受益人，無信託財產可供管

    5. 理、運用或處分時，受託人得逕行終止。
  5. (五) 因天災、不可抗力或其他不可歸責於甲乙雙方當事人之

    6. 事由，致受託人無法管理運用信託財產或信託行為無法
